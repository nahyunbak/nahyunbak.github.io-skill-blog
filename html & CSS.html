  1. 주축과 교차축

주축(main axis)

디폴트: 가로
속성(property): justify-content
값(value):  flex-start/flex-end/center/space-evenly..
property&value 참고 사이트: https://developer.mozilla.org/en-US/docs/Web/CSS/justify-content



교차축(cross axis)

디폴트: 세로
속성(property): align-items
값(value):  flex-start/flex-end/center/space-evenly..
property&value 참고 사이트: https://developer.mozilla.org/en-US/docs/Web/CSS/align-items




  1. flex 레이아웃의 주축의 방향 바꾸기

flex-direction

디폴트: 가로
속성(property): flex-direction
 값(value): row (디폴트) column column-reverse row-reverse
property&value 참고 사이트: https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction


첫 번째: flex-direction: row(디폴트)
(주축: 가로 // 교차축: 세로)




두 번째: flex-direction: row-reverse
(주축: 뒤집힌 가로 // 교차축: 세로)



세 번째: flex-direction: column
(주축: 세로 // 교차축: 가로)




네 번째: flex-direction: column-reverse
(주축: 뒤집힌 세로 // 교차축: 가로)




  1. 요소의 위치 바꾸기

flex-wrap

문제상황: 찌그러짐

첫 번째: flex-wrap: nowrap (디폴트)

(width나 height를 초기값으로 여김, 한 줄에 있게 만듬) 


두 번째: flex-wrap: wrap 

(width와 height값을 존중. 우측부터 아래줄로 넘어감. )  



세 번째: flex-wrap:  wrap-reverse 


(width와 height값을 존중. 우측부터 위엣줄로 넘어감. )

position




  1. position이란

언제 사용?:  미세하게 조정할 때 
어떻게? : 부모가 아닌, 자식(당사자)에게 position 속성을 넣어서 지시 
디폴트: position: static  == 박스를 처음 위치하는 곳에 둠 == 다른 요소(block, margin.. 등을 의식)


  1. position: fixed 

언제 사용?:  요소를 정지시켜놓고 싶을 때. 
어떻게? : 부모가 아닌, 자식(당사자)에게 'position: fixed' 속성 및 값을 넣어서 지시
활용1: 처음 만들어진 자리(다른 요소를 의식)에 고정되어 있다.
활용2: top, left, right, bottom 중 하나만 수정해도 다른 레이어에 위치하게 되어(다른 요소 무시) 원래 위치가 무시된다.(다른 요소 무시하니까)
활용3: 'position: fixed'를 사용하면  맨 앞에 위치한다.


 
(활용1)
예시1: 요소가 처음 만들어진 자리에 고정되어 있다.
